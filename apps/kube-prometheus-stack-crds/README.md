# kube-prometheus-stack custom resource definitions (CRDs)

- Major Updates vom Kube-Prometheus-Stack können neue CRDs mit sich bringen
- Helm upgrades aktualisieren by design keine CRDs
- Ob neue CRDs benötigt werden, sieht man hier: https://github.com/prometheus-community/helm-charts/blob/main/charts/kube-prometheus-stack/UPGRADE.md
